# AI-Based Anti-Money Laundering (AML) Transaction Monitoring System

## Project Overview

This project is developed as a Final Year B.Sc. Data Science project.

The objective of this project is to detect suspicious financial transactions using Machine Learning and provide explainable predictions using SHAP. The system also uses REST APIs for communication between the frontend, backend, machine learning model, and database.

---

## Problem Statement

Money laundering is a major challenge for banks and financial institutions. Millions of transactions occur every day, making it difficult to manually identify fraudulent or suspicious activities.

This project aims to build an intelligent AML system that automatically detects suspicious transactions using Machine Learning and provides explainable results for better decision-making.

---

## Objectives

- Detect suspicious transactions using Machine Learning.
- Explain predictions using SHAP.
- Store transaction data in PostgreSQL.
- Build REST APIs using FastAPI.
- Display results through a user-friendly dashboard.
- Generate investigation reports using an LLM (Ollama/Llama 3).
