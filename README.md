# AI-Based Anti-Money Laundering (AML) Transaction Monitoring System

## Final Year Project

### Project Description

This project aims to develop an AI-powered Anti-Money Laundering (AML) Transaction Monitoring System. The system will use Machine Learning to identify suspicious financial transactions, REST APIs to connect different components of the application, PostgreSQL for data storage, SHAP for explainable AI, and an LLM (Llama 3 via Ollama) to generate investigation reports.

---

## Technologies to be Used

- Python 3.11
- FastAPI
- PostgreSQL
- Scikit-learn
- SHAP
- Streamlit
- Ollama (Llama 3)
- Git & GitHub
- VS Code
